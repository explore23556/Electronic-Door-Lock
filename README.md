# Electronic-Door-Lock
## 一、專案摘要
### 此專案主要是模擬生活中常見的電子門鎖，並且透過NUC140開發板來實現相關的功能，而其專案流程圖如下所示，當使用者想要進入門時，會透過鍵盤上的按鍵來輸入密碼，若密碼輸入正確時，便打開門來讓人進入，反之，則會不斷地重新輸入密碼。
<p align="center">
<img src="https://github.com/explore23556/Electronic-Door-Lock/blob/main/%E9%9B%BB%E5%AD%90%E9%96%80%E9%8E%96.png"; width="60%";/>
</p>

## 二、我在專案中負責的工作
### 我主要是負責撰寫程式來讓使用者可以透過keyboard的輸入方式來輸入密碼，並且當密碼輸入正確時，會使NUC140開發板輸出訊號來控制馬達。
